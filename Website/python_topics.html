Python lists

What is Python list?

Python List is a item that you can store.

How to access, change, add and remove item in list?

You can access the list by using len()

You can change the list by changing the index.

You can remove the list by using remove()

You can add the list by using append()

How to sort a list?

You can sort a list using list.sort()

Selection Statements

if statement.

An if statement is a condition statement used to check a condition, and execute it if the condition holds true.

if-else statement

The if-else statement is used to execute both the true part and the false part of a given condition

if-elif statement

The “elif” keyword in Python, stands for “else if”. It can be used in conditional statements to check for multiple conditions
